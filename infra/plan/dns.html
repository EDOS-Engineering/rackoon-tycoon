1 · DNS for rackoon.cloud

Point the apex and www at the droplet's reserved static IP 164.90.247.182 (DigitalOcean reserved IP; visible on the Forge server detail page / DigitalOcean console).

Records to create

Create these at whatever hosts DNS for rackoon.cloud (registrar or DigitalOcean DNS if you delegate the nameservers there).

TypeHost / NameValueTTL
A@ (apex / rackoon.cloud)164.90.247.1823600
Awww164.90.247.1823600

Two A records pointing at the same IP keeps it simple and lets Forge issue one Let's Encrypt cert covering both rackoon.cloud and www.rackoon.cloud. We redirect www → apex in nginx (see Forge Setup).

TTL guidance

Use a low TTL (300–3600s) during cutover so mistakes propagate out fast. After go-live is verified you can raise the apex/www TTL to 3600+ for stability. There is no existing record to "lower in advance" here — the domain is freshly purchased and not yet pointed.

www as A vs CNAME

Either works for www:

  • A record (recommended here) — www → 164.90.247.182. One IP, both names, simplest.
  • CNAME alternative — www → rackoon.cloud. (note trailing dot). Valid because www is a subdomain. If the droplet IP ever changes you only update the apex A record.
Apex-CNAME caveat. You cannot put a plain CNAME on the apex (@ / rackoon.cloud) — DNS forbids a CNAME coexisting with the SOA/NS records that must live at the zone apex. So the apex must be an A record to the static IP (we have one, so this is fine). Some providers offer ALIAS/ANAME/"CNAME flattening" to fake apex-CNAME behaviour — unnecessary here since we have a stable static IP.

Optional: IPv6

If the droplet has a public IPv6 address, add matching AAAA records for @ and www pointing at it. Optional — skip if you don't need IPv6.

Verify propagation

dig +short rackoon.cloud        # expect 164.90.247.182
dig +short www.rackoon.cloud    # expect 164.90.247.182 (or the apex name if CNAME)

Wait until both resolve to the droplet IP before requesting the Let's Encrypt certificate in Forge — the HTTP-01 challenge needs the domain to already point at the droplet.
